## Exercice 1 :
On run la commande make dans le dossier hello qui va compiler le hello.c et le copier dans tous les dossier que l'on utilisera après pour faire nos dockerfile.

Ensuite on fait le dockerfile suivant :
```c
FROM ubuntu:22.04
WORKDIR /
COPY hello.exe .
CMD ["./hello.exe"]
```
